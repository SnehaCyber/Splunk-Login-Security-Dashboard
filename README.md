# Splunk-Login-Security-Dashboard
Splunk SIEM dashboard for monitoring windows login events and detecting brute force attacks
## Tools Used
- Splunk Enterprise
- Windows Event Logs
- SPL (Search Processing Language)
- VMware Workstation

## Events Monitored
- Event ID 4624 - Successful Login
- Event ID 4625 - Failed Login

## Dashboard Features
- Login activity monitoring
- Failed login analysis
- Brute force login detection
- Security alert visualization

## SPL Queries
Included searches for:
- Successful login monitoring
- Failed login tracking
- User-based login analysis

## Skills Demonstrated
- SIEM monitoring
- Log analysis
- Threat detection
- Security event investigation
